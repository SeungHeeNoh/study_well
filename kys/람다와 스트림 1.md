# [연습문제]
## [14-1] 메서드를 람다식으로 변환하여 아래의 표를 완성하시오.
### Q1) 
```  
int max(int a, int b) {
	return a > b ? a : b; 
}
```
### W1) 
```  
// 코드를 작성해주세요.
(int a, int b) -> a > b ? a : b
```
