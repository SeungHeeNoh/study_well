# [연습문제]
## [14-1] 메서드를 람다식으로 변환하여 아래의 표를 완성하시오.
### Q1) 
```  
int max(int a, int b) {
	return a > b ? a : b; 
}
```
### W1) 
```  
(int a, int b) -> a > b ? a : b;
```
----
### Q2) 
```  
void printVar(String name, int i) {
	System.out.println(name+"="+i);
}
```
### W2) 
```  
(String name, int i) -> { System.out.println(name+"="+i); };
```
----
### Q3) 
```  
int square(int x) {
	return x * x;
}
```
### W3) 
```  
x -> x * x;
```
----
### Q4) 
```  
int roll() {
	return (int)(Math.random() * 6);
}
```
### W4) 
```  
() -> (int)(Math.random() * 6);
```
----
### Q5) 
```  
int sumArr(int[] arr) {
	int sum = 0;
	for(int i : arr)
		sum += i;
	return sum;
}
```
### W5) 
```  
(int[] arr) -> {
	int sum = 0;
	for(int i : arr)
    	sum += i;
    return sum;
};
```
----
### Q6) 
```  
int[] emptyArr() {
	return new int[]{};
}
```
### W6) 
```  
() -> new int[]{};
```
